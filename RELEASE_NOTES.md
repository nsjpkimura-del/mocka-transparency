# MoCKA Insight System Release Notes

EN

This document summarizes notable changes across the MoCKA Insight System.
It is optimized for auditability: short entries, clear scope, and reproducible intent.

Release 2026-03-04

Scope
Ecosystem-wide documentation and hygiene hardening.

Highlights
- Canonical README redesign across repositories
- Architecture overview standardized and linked from README
- Backup and safe deployment workflow established
- LF canonicalization enforced and renormalization performed
- Community standards added (contributing, security, conduct, citation)
- Search optimization blocks added (keywords, verification quickstart)

Verification
- Use git log --show-signature to verify signed commits.
- Verify published hashes where applicable.
